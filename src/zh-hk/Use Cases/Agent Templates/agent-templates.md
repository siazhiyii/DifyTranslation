# 代理模板

## 什麼是代理模板？

代理模板是預先構建的配置，旨在幫助您快速創建針對特定任務量身定制的AI代理。這些模板作為發射台，提供基礎設置，您可以根據自己的獨特業務需求進行自訂。通過使用代理模板，您可以節省時間和精力，同時確保您的AI代理有效配置以實現其預期目的。

## 使用代理模板

有關如何使用代理模板的詳細說明，請參考 [從模板創建](/zh-hk/create-from-templates/)。本指南將指導您選擇、複製和自訂代理模板以滿足您的需求。
