# 使用AI代理

在完成設置後訪問和與您的AI代理互動！

## 訪問您的AI代理

一旦您配置並**保存**了AI代理的所有設置，您可以輕鬆地通過以下步驟訪問它：

1. 點擊位於AI代理構建器右上角的 **"打開圖標 ↗"** 按鈕。
2. 此操作將帶您進入您的AI代理的網頁應用程序或聊天機器人登陸頁面。

您現在已準備好使用您的AI代理並探索其功能！
